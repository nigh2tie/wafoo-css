メインコンテンツへスキップ

パターン1: ハンバーガーなし（デフォルト）

動作:
• デスクトップ（769px以上）: 横並び
• モバイル（768px以下）: 縦並びに自動変換
• JavaScriptなし、シンプルな実装
ブランド
  • ホーム
  • ドキュメント
  • お問い合わせ
<nav class="wf-navbar">
  <a class="wf-navbar-brand" href="#">ブランド</a>
  <ul class="wf-navbar-nav">
    <li class="wf-nav-item"><a class="wf-nav-link is-active">ホーム</a></li>
    <li class="wf-nav-item"><a class="wf-nav-link">ドキュメント</a></li>
  </ul>
</nav>

パターン2: ハンバーガーあり

動作:
• デスクトップ（769px以上）: 横並び、ハンバーガー非表示
• モバイル（768px以下）: ハンバーガー表示、サイドドロワー
• JavaScriptで開閉制御が必要
wafoo-css
  • ホーム
  • ドキュメント
  • お問い合わせ

  <a class="wf-navbar-brand" href="#">wafoo-css</a>

    <span class="wf-hamburger__line"></span>

    <span class="wf-hamburger__line"></span>
  </button>

  <ul id="nav-menu" class="wf-navbar-nav">
    <li class="wf-nav-item"><a class="wf-nav-link is-active">ホーム</a></li>
  </ul>
</nav>

パターン3: Primaryバリアント（ハンバーガーなし）

動作:
• テーマカラーの背景
• モバイルで縦並びに自動変換
ブランド
  • ホーム
  • ドキュメント
  • お問い合わせ

パターン4: Primaryバリアント + ハンバーガー

動作:
• テーマカラーの背景
• モバイルでサイドドロワー（テーマカラー継承）
wafoo-css
  • ホーム
  • ドキュメント
  • お問い合わせ

検証方法

  1. ブラウザの幅を768px以下にリサイズ
  2. パターン1: メニューが縦並びになることを確認
  3. パターン2: ハンバーガーが表示され、クリックでサイドドロワーが開くことを確認
  4. パターン3: Primaryカラーで縦並びになることを確認
  5. パターン4: Primaryカラーのサイドドロワーが開くことを確認
